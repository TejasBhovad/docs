# <Heading name="banner" description="Pattern 2">Python</Heading>

> Create the following pattern using Python

```
    *
   **
  ***
 ****
*****
```

```python
for i in range(1, 6):
    print(' ' * (6-i),end='')
    print('*' *( i))
```

:::note

To compile and run the program, you can use the following commands:

```bash
python3 foo.py
```

:::
