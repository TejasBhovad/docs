> Write a Python script that uses the Matplotlib library to create a simple line plot.

```python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 7, 11]

plt.plot(x, y)
plt.show()

# Output:
    # A simple line plot will be displayed.
```

```bash
python3 foo.py
```

:::
