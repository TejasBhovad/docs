# <Heading name="banner" description="Function overriding">Python</Heading>

> Implement function overriding in Python, where a child class provides its own implementation of a method that is already defined in the parent class.

```python
class Parent:
    def method(self):
        print("Parent Method")

class Child(Parent):
    def method(self):
        print("Child Method")

child = Child()
child.method()

# Output:
    # Child Method
```
