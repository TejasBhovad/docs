> Write a program to calculate area of circle, rectangle and square using function overloading

```python
class Area:
    def area(self, a, b=0):
        if b == 0:
             return a * a
        else:
            return a * b

area = Area()
print("Area of rectangle: ", area.area(5, 10))
print("Area of square: ", area.area(5))

```
