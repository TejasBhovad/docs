## Move Zeros from List to End

```python

list = [0,0,34,4,4,5,5]
len_list  = len(list)
new_list=[]
for i in list:
    if (i!=0):
        new_list.append(i)
for i in range(0,(len_list-len(new_list))):
    new_list.append(0)
```
